# SQL Text Replacement Techniques

This project demonstrates different ways to replace text values in SQL using
SELECT queries. It covers simple replacements, multiple replacements,
conditional logic, regex, and scalable solutions using mapping tables.

## Technologies
- SQL (MySQL / PostgreSQL compatible)

## Topics Covered
- REPLACE()
- Nested REPLACE()
- CASE expressions
- REGEXP_REPLACE
- Lookup / mapping tables

## Use Case
Useful for data cleaning, normalization, and reporting queries.
